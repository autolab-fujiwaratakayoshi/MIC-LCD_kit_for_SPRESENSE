### 2.MICを接続
アナログマイクですので、音声の録音、スルーに使用する通常はMIC_AとMIC_Bの２箇所に接続して使用してください
*４箇所接続すると録音やスルーが動作しないようです

FFTアナライザなどのアプリケーションの場合は４箇所接続してご使用いただくことも可能です
